Bases de Datos

Es la administración de diferentes sistemas de gestión de bases de datos independientemente de si es Oracle, SQLServer, MariaDB, MySql o PostgreSQL y logre comprender las diferentes sentencias DDL DCL y DML

El estudiante aprenderá a diseñar bases de datos independiente de su sistema manejador de base de datos para este objetivo se usara como primera instancia un MER (Modelo Entidad Relación) y como segunda instancia la cual ya es la diagramación real de su modelo de datos será el MR (Modelo Relacional).

El estudiante aprenderá acerca de los diferentes sistemas manejadores de bases de datos se familiarizará con las consultas básicas en SQL para así poder realizar uso de las estructuras básicas de control sobre la extracción y presentación de informes en una base de datos relacional

El estudiante aprenderá la diferencia y el cómo funcionan las diferentes sentencias DDL (Data Definition Language), DCL (Data Control Language) y DML (Data Manipulation Language).
